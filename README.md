# KC Dental Works (Vanilla HTML/CSS/JS + Express API)

Production-ready static site with a minimal Node/Express backend for forms.

## Quick Start
```bash
npm install
npm run dev
# Frontend at http://localhost:5000
# API at      http://localhost:4000
```

## Production
```bash
npm start
```

## Features
- WCAG 2.2 AA-friendly layout, skip link, focus-visible
- Sticky header, responsive nav, service cards, before/after reveal
- Contact + Appointment forms with client-side validation & honeypot
- API: POST /api/contact, POST /api/appointments (JSON storage)
- Design tokens and componentized CSS
- robots.txt + sitemap generator script
